**HZNUOJ 是基于 [HUSTOJ](https://github.com/zhblue/hustoj) 改造而来的，遵循GPL协议开源**

# 优势

* 更华丽的界面
* 更灵活的权限管理
* 支持多组样例
* 有封装好的docker镜像，更简单的部署
